Create Simple products

Use this page to add Simple products to WooCommerce. Check the products you want to add and select the action you want to use. When you select 'Add as new products' the products will be added to your WooCommerce. Use the column 'Existing product' to find an existing WooCommerce product for each selected product, then select 'Connect to existing products' to finish the action.
